Geometrical (matrix, quaternion, vector) extensions to JavaScript's "Math" module

### Installation
`npm install math`

### Modules

* `Matrix3`
* `Matrix4`
* `Vector2`
* `Vector3`
* `Quaternion`

### Features

* Supports browsers and node.js
* High-performance (allocation-free, native array)
* Written in CoffeeScript

Geometrical (vector, matrix, quaternion) extensions to JavaScript's "Math" module

# Modules

* `Vector2`
* `Vector3`
* `Matrix3`
* `Matrix4`
* `Quaternion`

# Features

* Works in browsers and node.js
* Allocation-free at run-time
* Uses native arrays rather than objects
* Written in CoffeeScript

# Installation
    npm install math
